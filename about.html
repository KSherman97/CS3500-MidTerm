Project Objective

This project covers basic HTML and CSS web design, and also utilizes Boostrap. The website is for a (fictitious) wine company that includes a catalog/online shop. The 5 HTML files are as follows:

Homepage (index page), about page (the current page), catalog page (product listing), visit us page, and a contact page.

The purpose of this project is to display fruitful knoweledge of various web technologies and frameworks including, but not limited to:

  • HTML
  • CSS
  • Bootstrap Framework
  • JavaScript

Site specifications

Initially the website was all hand designed with CSS, however it was decided that a more efficient way of developing a coherent and smooth flowing design would be to utilize the bootstrap framework. This would allow us to ensure that the website would be free-flowing and reactive to diferent screen sizes and viewports.
There were numerous other options for design frameworks - we did not pick bootstrap for any specific reason, other than it is simple to hit the ground running and quick to prototype designs. Boostrap also allowed us to quickly adapt new changes with minimal disruption to the rest of the site.
Unfortunately, because the site is not connected to a dedicated server, there is no way that we are aware of to send emails from the website. As an alternative proof of concept, we have created a JavaScript alert box popup with the first name and the email of the respective user as a confirmation.

All of the code can be found on GitHub.
